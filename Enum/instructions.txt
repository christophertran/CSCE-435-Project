1. compile RandomNumberGenerator.cpp
    g++ RandomNumberGenerator.cpp -o RandomNumberGenerator

2. Generate random numbers
    ./RandomNumberGenerator <HOW MANY RANDOM NUMBERS>

3. Load Intel module
    module load intel/2020b

4. Compile Algorithm
    mpiicc enum.c -o enum

5. Modify number of processes and random numbers binary in *mpi.grace_job* file

6. Submit batch job to grace cluster
    sbatch mpi.grace_job
